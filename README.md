# ts-react-redux-express-example

## Heroku App
https://ts-react-redux-express-example.herokuapp.com

## Install
```
$ yarn
```

## Debug
Run the following commands in several terminals.
```
$ yarn build:dev:server
```
```
$ yarn start:dev:server
```
```
$ yarn start:dev
```

## Build Production And Run
```
$ yarn start
```
